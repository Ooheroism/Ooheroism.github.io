我是如何写完一篇公众号文章的？

发表于 2024-01-05 更新于 2024-01-10 分类于 内容创作者 阅读次数：
本文字数： 1.2k 阅读时长 ≈ 2 分钟

今天，我爸打电话催更了，说昨天咋没发文章。作为我公众号的首席读者，我觉得我还是有必要向他介绍些我的创作过程，哈哈哈。不过这也是一个自我总结的过程，这是我写这篇文章的动机。
我的创作流程是在PARA（一种个人信息管理理念）的基础上，结合自己的体验改良过的版本。下面我会先介绍PARA的理念，然后介绍我改良后的版本，最后介绍我的创作过程。

1 PARA的理念介绍

PARA是一种个人信息管理的理念，它的全称是Projects-Areas-Resources-Archives，代表项目、领域、资源和档案，这四个顶级类别包括了你在工作和生活中可能遇到的所有类型的信息。

  • 项目（Projects）：明确要完成的具体目标，以及要达成这个目标的一连串任务和数据。
  • 领域（Areas）：自己目前有责任推进、关注的某个范围，例如想要学习的主题、想要改进的某种生活、某个职场的工作掌控等等。
  • 资源（Resources）：为了完成项目、推进领域需要的数据，或者自己长期感兴趣的各种参考数据。
  • 档案（Archives）：完成的项目、不再关注的领域、不再感兴趣的资源等等。

PARA的思想是，以完成一个又一个的项目为目标去记笔记。发布文章、视频、设计、PPT，组织亲友师生间的活动，甚至是给恋人的一个惊喜，都可以是项目。在开始一个项目的时候，先定好项目流程与子任务，然后直接从资源库中快速拿取知识、灵感或者方案，开始项目。在项目进行中时，又不断补充资料、任务等。
PARA的四种分类，不是硬性地把数据锁定在某个类目上，而是按信息“使用程度”来显示它们目前的“状态”。PARA里的信息是可以流动的：收集来的信息，还没有任何具体行动的，可以先放在资源里。對某类信息产生兴趣，打算长时间投注精力时，可以把它们置放在领域里。對此等兴趣有了明确的目标，要付诸实际行动时，就提升上项目。项目完成，便收藏于档案里，有需要时可以随时找到。

2 我对PARA理念的改良

  • P：Projects
    • 项目的基本要求是可以向别人展示，比如发在公众号的文章。
  • A：Areas
    • 领域的基本要求得让自己先理解，比如我画图的过程就是自己理解的过程。
  • R：Resources
    • 资源的基本要求是我自己读得下去、读得明白，否则就谈不到对领域的理解了。
  • A：Archives
    • 存档的要求是方便之后检索，就是之后回顾的时候可以找得到，找得快。

3 我的创作过程

  • 从目标开始：
    • 比如这篇文章，我爸催更，所以觉得有必要解释下我的创作过程。那就先在图片库找找相关图片，找到的就是这张PARA图。
    • 这张图基本可以帮我表达清楚我的目标——创作流程介绍了，所以就不去找新资料了。
    • 假如没找到图，或找到的图不足以解释我的创作过程，那就去得读资料了，那这篇稿子也就暂时出不来了。
  • 从资源开始：
    • 比如我学习了李松蔚的《心理咨询》课（他毕业于北大心理学系），自己理解的过程中，画了好多图。
    • 学习过程中意外发现，其中心理咨询师，“促成改变”的观念正好可以帮我解释某个疑问。
    • 那就将这个疑问和我找到的解释，以发文的形式向大家展示。
